# Data

This repository analyzes publicly available transcriptomic datasets. To comply with the data providers' distribution policies and GitHub file size limitations, the original datasets are **not redistributed** in this repository.

## Human Single-Cell RNA Sequencing

**Study:** Accelerating Medicines Partnership (AMP) Lupus Nephritis Consortium

- **ImmPort Study:** SDY997
- **Species:** Human
- **Technology:** Single-cell RNA sequencing
- **Purpose:** Discovery of inflammatory macrophage-associated therapeutic targets

---

## Mouse Single-Cell RNA Sequencing

**Dataset:** GSE201932

- **Species:** Mouse
- **Technology:** Single-cell RNA sequencing
- **Purpose:** Cross-species validation of candidate therapeutic targets

---

## Human Spatial Transcriptomics

**Dataset:** GSE263909

- **Species:** Human
- **Technology:** Spatial transcriptomics
- **Purpose:** Independent spatial validation of prioritized targets

---

## Processed Files

Intermediate analysis outputs generated by the workflow (e.g., marker tables, target rankings, and summary results) are provided in the `results/` directory.

Large processed objects (e.g., Seurat objects) are intentionally excluded because they exceed GitHub file size limits.

---

## Reproducing the Analysis

1. Download the original public datasets listed above.
2. Place the downloaded files in the appropriate directories described in the project documentation.
3. Run the analysis scripts in the order described in the main `README.md`.

Following these steps will reproduce all intermediate results and publication-quality figures generated in this repository.
