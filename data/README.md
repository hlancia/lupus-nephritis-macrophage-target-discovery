## Processed Files

The processed intermediate files generated by the analysis are available in the `results/` directory.

Large processed datasets (e.g., Seurat objects) are intentionally excluded because they exceed GitHub file size limits.

Users should download the original public datasets and reproduce the analysis using the scripts provided in this repository.
