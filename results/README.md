# Results

This directory contains the intermediate and final outputs generated by the computational workflow for cross-species therapeutic target discovery in lupus nephritis.

The results document each stage of the analysis, from inflammatory macrophage identification to the final consensus ranking of therapeutic targets.

---

## Result Files

| File | Description |
|------|-------------|
| **AMP_LN_all_markers.csv** | Marker genes identified for all cell clusters in the human AMP Lupus Nephritis single-cell dataset. |
| **AMP_LN_cluster_markers.csv** | Differentially expressed genes for each annotated cell cluster. |
| **AMP_LN_cell_annotations.csv** | Cell identities assigned to each single cell following clustering and annotation. |
| **Inflammatory_Macrophage_markers.csv** | Differentially expressed genes enriched in inflammatory macrophages used for candidate target discovery. |
| **Macrophage_Targets.csv** | Initial list of candidate macrophage-associated therapeutic targets identified from the human discovery dataset. |
| **Cross_species_target_validation_table.csv** | Validation of candidate targets using the independent mouse lupus nephritis single-cell RNA sequencing dataset. |
| **Consensus_Target_Ranking.csv** | Integrated evidence matrix summarizing support from human single-cell RNA sequencing, mouse validation, spatial transcriptomics, and macrophage spatial correlation analyses. |
| **Final_Target_Ranking.csv** | Final prioritized list of therapeutic targets based on the consensus evidence-scoring framework. |

---

## Analysis Workflow

The files are generated sequentially throughout the computational pipeline:

```text
Human scRNA-seq
        │
        ▼
Cell Annotation
        │
        ▼
Differential Expression
        │
        ▼
Candidate Target Identification
        │
        ▼
Cross-Species Validation
        │
        ▼
Spatial Validation
        │
        ▼
Consensus Target Ranking
```

---

## Final Output

The primary output of this project is:

- **Final_Target_Ranking.csv**

This file contains the final prioritized therapeutic targets supported by multiple independent lines of biological evidence and represents the principal result of the computational workflow.

## Reproducibility

All files in this directory are generated directly from the R analysis scripts included in the repository.

No manual editing of analysis results was performed.
